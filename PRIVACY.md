## 2. Data Collection & Usage

### 2.1. Analytics (Vercel)

The live application uses **Vercel Analytics** to collect anonymous usage statistics.

- **Purpose:** To analyze traffic trends and optimize website performance.
- **Data Collected:** Anonymous page views, browser type, operating system, and country of origin.
- **Privacy:** No personally identifiable information (PII) is collected. No cookies are used for this purpose.

### 2.2. Contact Form (EmailJS)

When visitors use the **"Contact Me"** form, the data they enter is processed via **EmailJS**.

- **Data Collected:** Name, Email Address, Message content.
- **Purpose:** Solely to deliver the message to the portfolio owner's inbox.
- **Storage:** Data is not stored in a persistent marketing database.

### 2.3. Local Storage

This application uses the browser's `localStorage` API.

- **Purpose:** To remember the user's preference for Dark Mode or Light Mode across sessions.
- **Data Transmission:** This data remains strictly on the user's device and is never sent to a server.
